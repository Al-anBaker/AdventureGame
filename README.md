# Map Legend:
* "@" is the Player.
* "%" is an Enemy.
* "░" is an Empty Space.
* "█" is an unmoveable wall.
* "/" is a Dagger.
* "!" is a Health Potion
* "]" is Leather Armour
* "~" is Water.
* ">" is a Door.
