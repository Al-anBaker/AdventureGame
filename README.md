# AdventureGame
This is a Python CLI based Rouge-Like tech demo.
You control the Player with WASD using prompts in the console

# Map Legend:
* "@" is the Player.
* "%" is an Enemy.
* "░" is an Empty Space.
* "█" is an unmoveable wall.
* "C" is a Chest.
* "~" is Water.
